#  Configuration 정보

## Configuration은 총 4개의 파일이 존재한다.
- Config Docs
- GoogleService-Info.plist
- Debug.xcconfig
- Release.xcconfig

### Config Docs
- 현재 보고 있는 파일

### GoogleService-Info.plist
- Firebase를 사용하기 위해 필요한 파일

### Debug.xcconfig, Release.xcconfig
- Debug / Release 빌드를 나누기 위해 존재하는 파일

### 각 파일이 담고있는 Key-Value

- App Name
- Base URL
- Kakao App Key
